# SEO Strategy Guide

This document outlines the SEO strategy and capabilities integrated into the **Gaime-Bridge** project through our automated agent skills.

## Core Pillars

### 1. Technical Excellence
We prioritize **Core Web Vitals** and technical infrastructure to ensure a smooth, fast experience for both users and search engine bots.
- **Tools**: `seo-technical` skill.
- **Focus**: Performance (LCP, INP, CLS), HTTPS, Mobile-first design, and clean indexing (Robots/Sitemap).

### 2. Semantic Understanding (Schema)
We use **JSON-LD Structured Data** to provide clear context to search engines, enabling rich results and better ranking for software-related queries.
- **Tools**: `seo-schema` skill.
- **Focus**: Organization, SoftwareApplication, and WebSite markup.

### 3. Content Authority (E-E-A-T)
We adhere to Google's **Experience, Expertise, Authoritativeness, and Trustworthiness** guidelines.
- **Tools**: `seo-content` skill.
- **Focus**: Professional developer content, clear author identity, and high-trust signals (security/transparency).

## How to use the SEO Agent

You can ask me to perform various SEO tasks by referencing these skills. For example:
- *"Run a full SEO audit on the operator page."*
- *"Check the technical SEO of our landing page."*
- *"Generate SoftwareApplication schema for this project."*
- *"Evaluate my latest blog post for E-E-A-T."*

## Reporting

All SEO audits will produce a prioritized report categorized by:
- 🔴 **Critical**: Fix immediately (affects indexation/ranking).
- 🟡 **Warning**: Improvement needed for optimal performance.
- 🟢 **Opportunity**: Strategic enhancements for better visibility.
